## Summary
**Movie Website** – A fully responsive **React.js movie web application** for browsing, searching, and exploring popular and latest movies. Built with **Bootstrap** for a responsive grid layout and **Axios** for dynamic API data fetching. Ideal for showcasing movie collections with a modern and interactive interface.

## Overview
- Developed with **React.js** to create a dynamic and interactive movie showcase platform.  
- **State management** handled using `useState` and `useEffect` for interactive UI updates and data lifecycle control.  
- **Axios API integration** to fetch and display movie information dynamically.  
- **Responsive layout** implemented using Bootstrap to ensure compatibility across desktop, tablet, and mobile devices.  
- Built reusable and engaging components to enhance user experience.
