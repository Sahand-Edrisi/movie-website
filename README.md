## Overview
Responsive movie showcase website built with **React.js**.  
- State management handled using `useState` and `useEffect` for interactive UI updates and data lifecycle control.  
- Integrated API requests via **Axios** to fetch and display movie data dynamically.  
- Responsive layout and grid system implemented with **Bootstrap** for cross-device compatibility.  
- Built dynamic and engaging components to enhance user experience.
